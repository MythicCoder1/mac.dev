About

Mac.dev started out as a 5th grade school project. The project was called a "Passion Project" where students had to pick one of their passions. I wanted to learn more about web dev, so I created this site. Honestly, I got wrapped up in web development and now this site is a lot bigger than orignally intended.

I created this website for anyone young person (or any old person, for that matter) to learn the fundamentals of CSS, HTML, and frontend JavaScript. Also, this website was NOT built with Wix, Wordpress, or any other website builder. It was bulit in Apple TextEdit with CSS and HTML. Over 200 lines of code were written to get this far.
