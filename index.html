CSS & HTML

Getting Started With HTML and CSS

All Macs have a built in code editor called TextEdit. To access it, go to your search bar in the top right corner of your screen. Type in "TextEdit" and click the top result. Once there, click the TextEdit Preferences and click plain text, then unclick smart quotes. To test it, write: "Hello, World". Save the file as "Hello.html" and then go into Finder. Right click on the HTML file and open in Chrome. If all goes well, it should say "Hello, World" at the top left part of your screen. Make sure to always end your document with ".html".
